## What is this?

This guide helps beginners (including kids!) discover programming by using AI coding assistants.  
Instead of writing every line yourself, you describe what you want in plain language, and an AI agent writes and runs the code for you — for free.

**Tools used:**
- [OpenCode](https://opencode.ai) — AI coding agent that runs in your terminal
- [NVIDIA Build API](https://build.nvidia.com) — Free access to powerful AI models
- [Playwright MCP](https://github.com/microsoft/playwright-mcp) — Lets the AI control a web browser
- [GitHub](https://github.com) — Free hosting for your project's source code

## Qu'est-ce que c'est ?

Ce guide aide les débutants (même les enfants !) à découvrir la programmation grâce aux assistants IA.  
Au lieu d'écrire chaque ligne soi-même, on décrit ce qu'on veut en langage naturel, et un agent IA écrit et exécute le code à notre place — gratuitement.

**Outils utilisés :**
- [OpenCode](https://opencode.ai) — Agent IA de développement qui fonctionne dans le terminal
- [NVIDIA Build API](https://build.nvidia.com) — Accès gratuit à des modèles IA puissants
- [Playwright MCP](https://github.com/microsoft/playwright-mcp) — Permet à l'IA de contrôler un navigateur web
- [GitHub](https://github.com) — Hébergement gratuit pour le code source de votre projet
